---
title: Accessing form event
description: Accessing form events on Unform
---

On the web, you can access the form event in the 3rd parameter of the submit
function.

```jsx lineNumbers=true
export default function MyForm() {
  function handleSubmit(data, { reset }, event) {
    console.log(event)
  }

  return <Form onSubmit={handleSubmit}>...</Form>
}
```
